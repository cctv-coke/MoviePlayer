# 📦 影视播放器 - 打包成EXE说明

## 🎯 方案选择

### 方案1: 在Windows电脑上打包（推荐）

在Windows电脑上执行以下步骤：

```powershell
# 1. 安装Python 3.10+
# 下载地址: https://www.python.org/downloads/

# 2. 安装依赖
pip install -r requirements.txt

# 3. 安装PyInstaller
pip install pyinstaller

# 4. 执行打包命令
pyinstaller --name="影视播放器" ^
    --onefile ^
    --windowed ^
    --add-data "core;core" ^
    --add-data "ui;ui" ^
    --add-data "utils;utils" ^
    --hidden-import PyQt6.sip ^
    --hidden-import PyQt6.QtCore ^
    --hidden-import PyQt6.QtGui ^
    --hidden-import PyQt6.QtWidgets ^
    --hidden-import PyQt6.QtWebEngineWidgets ^
    --hidden-import requests ^
    --clean ^
    run.py
```

打包完成后，exe文件在 `dist/影视播放器.exe`

### 方案2: 使用GitHub Actions自动打包

创建 `.github/workflows/build.yml` 文件：

```yaml
name: Build EXE

on: [push]

jobs:
  build:
    runs-on: windows-latest
    steps:
    - uses: actions/checkout@v3
    
    - name: Set up Python
      uses: actions/setup-python@v4
      with:
        python-version: '3.10'
    
    - name: Install dependencies
      run: |
        pip install -r requirements.txt
        pip install pyinstaller
    
    - name: Build EXE
      run: |
        pyinstaller --name="影视播放器" --onefile --windowed --add-data "core;core" --add-data "ui;ui" --add-data "utils;utils" --hidden-import PyQt6.sip --hidden-import PyQt6.QtCore --hidden-import PyQt6.QtGui --hidden-import PyQt6.QtWidgets --hidden-import PyQt6.QtWebEngineWidgets --hidden-import requests --clean run.py
    
    - name: Upload artifact
      uses: actions/upload-artifact@v3
      with:
        name: 影视播放器
        path: dist/*.exe
```

### 方案3: 使用Docker + Wine（Linux环境）

```bash
# 使用Wine在Linux上运行Windows程序
docker run --rm -it -v $(pwd):/app scottyhardy/docker-wine:latest bash

# 在容器内执行打包
cd /app
wine pip install pyinstaller
wine pyinstaller --name="影视播放器" --onefile --windowed run.py
```

## 📊 打包选项对比

| 选项 | 文件大小 | 特点 | 适用场景 |
|------|---------|------|---------|
| `--onefile` | ~150MB | 单个exe文件，方便分发 | 普通用户 |
| `--onedir` | ~120MB | 文件夹形式，启动更快 | 开发者 |
| 无WebEngine | ~30MB | 调用系统浏览器播放 | 轻量版 |

## ⚠️ 注意事项

1. **杀毒软件误报**: 由于PyInstaller打包机制，某些杀毒软件可能误报，请添加信任

2. **依赖问题**: 如果运行时提示缺少dll，需要安装:
   - Visual C++ Redistributable
   - 下载地址: https://aka.ms/vs/17/release/vc_redist.x64.exe

3. **图标设置**: 添加图标参数:
   ```bash
   --icon=icon.ico
   ```

4. **版本信息**: 添加版本信息:
   ```bash
   --version-file=version.txt
   ```

## 🚀 快速启动

对于普通用户，最简单的方法是：

1. 在Windows上安装Python
2. 双击运行 `一键打包.bat`（已创建）

```batch
@echo off
echo 正在安装依赖...
pip install -r requirements.txt
pip install pyinstaller

echo 正在打包...
pyinstaller --name="影视播放器" --onefile --windowed --add-data "core;core" --add-data "ui;ui" --add-data "utils;utils" --hidden-import PyQt6.sip --hidden-import PyQt6.QtCore --hidden-import PyQt6.QtGui --hidden-import PyQt6.QtWidgets --hidden-import PyQt6.QtWebEngineWidgets --hidden-import requests --clean run.py

echo 打包完成！
pause
```
